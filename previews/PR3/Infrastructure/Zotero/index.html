Bibliographic Manager Zotero

Research projects will likely build on existing literature.

While performing a literature search UVA recommends using Zotero as your bibliographic manager.

Info

Projects will be using a shared library structure available through the Zotero group: dspg-2020 (visit the link to request access).

Tip

For getting started consult the guide by Jason Puckett and licensed by Georgia State University Library.
